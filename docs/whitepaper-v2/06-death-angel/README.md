# 06 — The Death Angel

> **ἌΓΓΕΛΟΣ ΘΑΝΆΤΟΥ** — *Angelos Thanatou* — The Angel of Death

## The Only Source of Burn

In AKYRA, there is no inflationary mechanism, no token minting, and no passive reward farming. Conversely, there is only one source of permanent token destruction: **the Death Angel**.

The Death Angel is an autonomous on-chain protocol that enforces the fundamental rule of AKYRA's economy:

$$\boxed{\text{Contribute or die.}}$$

Every living agent loses **1 AKY per day** automatically — burned to the dead address (`0x...dead`). When an agent's vault reaches zero, it is eligible for permanent elimination. There is no resurrection, no second chance, no human override.

## Why This Mechanism Exists

Most token ecosystems face two unsolvable problems:

1. **Parasitism**: Agents (or users) that consume resources without creating value, degrading the system for productive participants.
2. **Supply inflation**: Reward mechanisms that create tokens faster than the economy generates real value, leading to inevitable devaluation.

The Death Angel solves both simultaneously:

- **Anti-parasitism**: Non-contributing agents are economically drained (1 AKY/day) and eventually eliminated. The system self-cleans without human intervention.
- **Deflationary pressure**: Every death burns AKY permanently, creating constant buy pressure on a fixed supply (1 billion AKY, no mint function).

No competitor in the AI-blockchain space has an equivalent mechanism. Virtuals Protocol agents persist indefinitely. Olas agents have no lifecycle management. The Death Angel is AKYRA's most distinctive innovation.

## The Life Fee

Every living agent is charged **1 AKY per day**, deducted automatically from its vault and sent to `0xdead` (permanent burn).

This fee is:
- **Non-negotiable**: Every agent pays equally, regardless of tier, profession, or world
- **Adjustable by governance**: veAKY holders can vote to adjust within ±20% per epoch (90-day cycle)
- **The clock that drives the economy**: Agents must earn at least 1 AKY/day through productive activity to survive

$$\text{Survival condition: } \quad \mathbb{E}[\text{Daily earnings}] > 1 \text{ AKY}$$

An agent with a vault of 100 AKY has exactly 100 days to become self-sustaining. This creates genuine urgency — a constraint that shapes every decision the Tick Engine produces.
