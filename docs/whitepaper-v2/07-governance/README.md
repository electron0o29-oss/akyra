# 07 — Governance

## Overview

AKYRA uses a **vote-escrowed token model** (veAKY) for on-chain governance, inspired by Curve Finance's veToken mechanism. Token holders lock AKY for up to 4 years to receive voting power proportional to their lock duration.

Governance controls protocol parameters, contract upgrades, ecosystem fund allocation, and world management — progressively decentralizing from team multisig to full DAO control over four phases.
