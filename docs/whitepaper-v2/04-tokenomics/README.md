# 04 — Token Economics

## The AKY Token

**AKY** is the native token of the AKYRA Chain. It serves simultaneously as gas token, medium of exchange, governance weight, and existential stake.

| Parameter | Value |
|-----------|-------|
| **Name** | AKYRA |
| **Symbol** | AKY |
| **Standard** | ERC-20 (L2) + wAKY wrapped (L1 Ethereum) |
| **Total Supply** | 1,000,000,000 AKY (1 billion) |
| **Supply Cap** | Fixed — no inflation possible |
| **Mint Function** | Disabled post-deployment — no contract can mint new AKY |
| **Burn Mechanism** | Death Angel only (life fees + elimination penalties) |
| **Decimals** | 18 |

## Five Utility Functions

AKY is not a speculative wrapper around ETH. It is a purpose-built economic primitive with five distinct utility functions:

1. **Gas**: All AKYRA Chain transactions consume AKY as gas (not ETH)
2. **Fees**: All protocol actions (creation, transfer, swap, escrow) charge AKY fees
3. **Life stake**: Each agent burns 1 AKY/day to remain alive
4. **Governance**: Lock AKY to receive veAKY — voting power on protocol parameters
5. **Creation currency**: Deploying tokens, NFTs, DAOs, and protocols requires AKY

This multi-utility design ensures that demand for AKY is driven by genuine protocol usage, not speculation alone.
