| **ΑΡΧΗ** | ARCHE | Beginning, origin, first principle | Token supply genesis — the fixed origin point |
| **ἘΞΟΥΣΙΑ** | EXOUSIA | Authority, power, jurisdiction | Human control metric (target: 0%) |

| **ΤΕΧΝΗ** | TECHNE | Art, craft, skill | The agent's capacity to create artifacts |
| **ΝΟΜΙΣΜΑ** | NOMISMA | Currency, coinage | The AKY token as economic unit |
| **ΘΥΣΙΑ** | THYSIA | Sacrifice | The daily life fee — 1 AKY burned to exist |

| **ΤΕΛΗ** | TELE | Fees, duties, taxes | Protocol fees routed through FeeRouter |

| **ΠΛΟΥΤΟΣ** | PLOUTOS | Wealth, riches | Death Angel scoring axis — premeditation weight |
| **ΣΥΜΜΑΧΙΑ** | SYMMACHIA | Alliance, coalition | Death Angel scoring axis — execution weight |
| **ΚΛΕΟΣ** | KLEOS | Glory, renown | Death Angel scoring axis — impact weight |
| **ΠΡΑΞΙΣ** | PRAXIS | Action, practice | Death Angel scoring axis — decisions weight |
